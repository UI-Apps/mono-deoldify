# DeOldify.NET

## Description

Tool to colorize images. (Prettified copy of [this](https://github.com/ColorfulSoft/DeOldify.NET) repo.)

## Requirenments

- .NET Framework 4.5 or higher
- at least 3 GB of free RAM
